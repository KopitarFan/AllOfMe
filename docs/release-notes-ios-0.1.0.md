# All Of Me iOS 0.1.0 TestFlight Notes

All Of Me is a local-first system tracking app. This first TestFlight build focuses
on private on-device setup and core daily workflows.

## Included

- First-run setup for starting fresh, importing a backup, or loading demo data.
- Local system profile, member, and group management.
- Member profile images from the device photo library.
- Fronting toggles, fronting sessions, timeline notes, and local insights.
- Backup export/import using JSON.
- Recently Deleted recovery for archived members/groups and deleted timeline entries.
- Optional app lock with Face ID or device passcode.
- Clear all local data for the current device.

## Known Limits

- Sync is not included in this build.
- Sharing/collaboration is not included in this build.
- Web still uses browser-safe local preferences while mobile uses SQLite/Drift.
