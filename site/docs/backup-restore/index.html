Backup and restore

Readable backups, handled carefully.

All Of Me can export and import local JSON backups. These files are useful for portability, and they should be treated as sensitive.

Exporting a backup

A local backup is created only when you choose to export. The exported file is controlled by you after export, including where it is saved, shared, copied, or deleted.

Importing a backup

Importing a backup restores data from the selected file onto the current device. Review the backup source carefully before import, especially if the file has been copied through email, cloud storage, or another app.

What backups may contain

  • System profile details.
  • Members, roles, notes, colors, groups, and archived state.
  • Fronting state, fronting sessions, timeline notes, and recently deleted items.
  • Profile images included for portability.

Storage tips

  • Store backups somewhere you trust.
  • Delete old backups from places where you no longer want them stored.
  • Avoid sending backup files to support unless you intentionally choose to share them.
  • Create a fresh backup before clearing local app data or replacing a device.
