New device guide

Moving phones should not mean starting over.

If you already connected Cloud Save, use an existing connected device to link the new device, then restore with your recovery key.

If you still have an existing connected device

  1. Open All Of Me on the existing connected device.
  2. Open Settings and go to Cloud Save.
  3. Create a one-time device link code.
  4. Install All Of Me on the new device.
  5. Choose the option to link an existing Cloud Save device.
  6. Enter the link code on the new device.
  7. Restore the latest Cloud Save package using the recovery key used when saving.

If you have a local backup file

You can import a readable JSON backup directly into the new device. Treat backup files as sensitive, because they contain the app data included at export time.

If you do not have an existing device or backup

Support may not be able to restore data if every connected device is unavailable, no exported backup exists, no Cloud Save restore point exists, or the recovery key is lost. All Of Me Cloud is not designed to decrypt encrypted restore points.

Before replacing a phone

Create a fresh backup or Cloud Save restore point, verify that you have the recovery key, and keep at least one connected device available until the new device is restored.
